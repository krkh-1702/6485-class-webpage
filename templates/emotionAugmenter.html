emotionAugmenter is a real-time avatar emoji system that updates based on the emotion of the text conversation. Using Natural Language Processing (NLP) with Python, we detecte the emotions of a conversation and select the highest percent option. Then we used that text to replace the target word for StyleCLIP facial expression generation, finally replacing a neutral facial expression with the updated avatar based on emotion.
